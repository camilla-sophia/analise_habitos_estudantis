# Dashboard: Hábitos de Estudo e Desempenho Acadêmico

Projeto de análise de dados sobre hábitos de estudo e desempenho acadêmico de alunos, com dashboard interativo desenvolvido em Streamlit.

## Objetivo

Explorar a relação entre hábitos de estudo (horas de estudo, frequência, sono) e características pessoais (gênero, escolaridade dos pais, trabalho de meio período) com o desempenho final dos alunos.

## Dataset

1000 registros com informações sobre hábitos de estudo, características demográficas e notas de alunos.

## Etapas do projeto

- Limpeza de dados (tratamento de valores faltantes)
- Transformação de variáveis categóricas (one-hot encoding e codificação ordinal)
- Análise exploratória com visualizações (distribuições, dispersão, correlação)
- Dashboard interativo com filtros dinâmicos (Streamlit + Plotly)

## Principais descobertas

- Horas de estudo diário é o fator com maior correlação com a nota final (0.57)
- Nota anterior do aluno também tem correlação relevante (0.41)
- Frequência às aulas tem correlação fraca a moderada (0.26)
- Escolaridade dos pais mostra diferença discreta, mais visível em níveis mais altos (PhD)
- Horas de sono, gênero e atividades extracurriculares não apresentaram influência relevante nessa amostra

## Tecnologias utilizadas

- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Streamlit, Plotly

## Como executar

```bash
pip install -r requirements.txt
streamlit run app.py
```
